## Talk Title

### Building Real-Time AI Voice Agents

## Abstract

Step into the developer’s kitchen and discover how to build AI voice agents that deliver fast and engaging conversations using Twilio. This session will cover code recipes for creating voice agents that listen and respond naturally and handle interruptions smoothly. You will learn how to select the best tools for building voice agents, optimize for speech latency, and enhance the user experience with effective voice prompt design. Unlock practical techniques and secret ingredients for building conversational AI that delights users and makes every conversation memorable.
